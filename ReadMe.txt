Node.js needs to be installed for this to function.
storage for server is hosted on mongoDB

open the node.js command prompt and navigate to this folder
run npm install
this will install dependencies need to run server
once installed run node server
the server will be hosted on localport 8080

as of now there is some functionality from within the browser, but the easiest way to test the API is through postman
https://www.getpostman.com/

Server uses: Node.js, mongoose, expressjs, and Jade.
When testing with postman make sure to enable x-www-form-urlencoded when trying to POST
